# What is this repository?

Random value generator with Kotlin.

# Example.

```kotlin
fun example() {
    Random.int8()       // signed 8bit
    Random.string(32)   // 32 chars.
}
```

# How to Install

```groovy
// build.gradle
dependencies {
    implementation 'io.github.eaglesakura.static-random:static-random:${replace version}'
}
```
